# IslasEnGuerra_ProyectoC
⚔️ Proyecto Final: Islas en Guerra (Simulador de Estrategia por Turnos en C)

 Estudiante 1: Diseñador del Mapa y Exploración	
 Responsabilidad principal: Creacion y visualizacion de los mapas y la logica de movimiento/descubrimiento 
 Conceptos Aplicadios: Arrays Bidimensionales, punteros. 
 Archivos modulares: mapa/mapa.c y mapa/mapa.h

 Estudiante 2: Gestor de Recursos y Unidades
 Responsabilidad principal: Administración de recursos (oro, madera, comida) y el sistema de entrenamiento/costos de las unidades militares.
 Conceptos Aplicadios: Arreglos de Estructuras, Punteros, Gestión Dinámica de Memoria.
 Archivos modulares: recursos/recursos.c y recursos/recursos.h

 Estudiante 3: Sistema de Batalla y Archivos
 Responsabilidad principal: Simulación de batallas por atributos y la persistencia de datos (guardado/carga) del juego.
 Conceptos Aplicados: Archivos Binarios (.bin) y de Texto (.txt), Lógica de Combate.
 Archivos modulares: sistema/sistema.c y sistema/sistema.h
